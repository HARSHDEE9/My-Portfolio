# Harshdeep Portfolio

An interactive multi-page personal portfolio built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` - landing page
- `about.html` - profile and skills
- `projects.html` - project showcase
- `journey.html` - education, certifications, and activities
- `resume.html` - embedded PDF resume with download support
- `contact.html` - contact links

## Features

- Multi-page portfolio structure
- Custom visual design with gradients and animated background
- Scroll-based reveal animations
- Mobile-friendly responsive layout
- Embedded resume preview
- Easy to host on any static hosting platform

## Run Locally

Open the folder in a browser directly, or use a local server for the best experience.

### Using Python

```bash
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/index.html
```

## Customize Content

- Update text inside the HTML files
- Replace `assets/resume/harshdeep-resume.pdf` with your latest resume
- Edit colors, layout, and typography in `assets/styles.css`
- Edit interactions in `assets/app.js`

## Deploy

You can deploy this site on:

- Vercel
- Netlify
- GitHub Pages

After deployment, connect your personal domain from your domain provider by updating DNS records in the domain dashboard.
